# Handwritten Digit OCR (Interactive Neural Network)

A simple interactive OCR demo where you can draw digits and train a small neural
network in real time. The backend is powered by Python, and the frontend uses an
HTML canvas for digit input.

## How to Run
1. Start the backend server:
2. Open `ocr.html` in your browser.
3. Draw digits, label them, and train the model interactively.

## Features
- HTML canvas digit input
- Real-time neural network training
- Lightweight Python HTTP backend

## Technologies
- Python
- JavaScript
- HTML Canvas
